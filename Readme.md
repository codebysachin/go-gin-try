## Clone and Run

To clone and run this project, follow these steps:

1. Clone the repository:
    ```
    git clone https://github.com/codebysachin/go-gin-try.git
    ```

2. Navigate to the project directory:
    ```
    cd go-gin-try
    ```

3. Build and run the project:
    ```
    go run main.go
    ```
